### 树形结构

很多评论模式其实就是树形结构的一种简化。这里我们可以参考[SQL反模式](https://book.douban.com/subject/6800774/)里面所说的做法

- 邻接表：依赖`parent_id`

```sql
create table comment
(
  id          serial not null constraint comment_pkey primary key,
  parent_id   integer,
  user_uid    interger,
  content     text
);
```
